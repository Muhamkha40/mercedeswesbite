Mercedes-Benz C-Class

Common Model Names (Variants)

  • C 180 / C 200 / C 300 → petrol or mild-hybrid engines
  • C 220d / C 300d → diesel options (common in Europe)
  • C 43 AMG / C 63 AMG → high-performance models

Key Features

  • 11.9” touchscreen with MBUX infotainment
  • Mild-hybrid technology in newer models
  • High-quality interior with ambient lighting
  • Adaptive cruise control, lane keep assist, parking assist
  • Optional 4MATIC (all-wheel drive)
  • AMG Line trims with sportier styling

Suitable for: daily driving with comfort + sporty feeling.

Mercedes-Benz E-Class

Common Model Names (Variants)

  • E 200 / E 300 / E 450 → petrol or mild-hybrid
  • E 220d / E 300d → diesel options
  • E 53 AMG → performance version
  • E 300e / E 400e → plug-in hybrid models

Key Features

  • Dual widescreen digital cockpit (MBUX 2.0)
  • Advanced driver assistance system (ADAS)
  • Air suspension for extra comfort (optional)
  • Voice control with “Hey Mercedes”
  • High-end Burmester® 3D sound system
  • Ambient lighting with 64 colors
  • Heated and ventilated seats

Suitable for: luxury + comfort with smooth long distance driving.

Mercedes-Benz G-Class (G-Wagon)

Common Model Names (Variants)

  • G 500 / G 550 → V8 petrol engines
  • G 400d → diesel (Europe)
  • AMG G 63 → high-performance flagship

Key Features

  • Legendary boxy design
  • 4MATIC all-wheel drive standard
  • 3 locking differentials → serious off-road ability
  • 360° camera system
  • Air suspension + adjustable driving modes
  • Nappa leather, ambient lighting, digital cockpit
  • Optional Maybach interior pack (ultra luxury)

Suitable for: those who want power, prestige, and off-road ability.

Mercedes-Benz S-Class

Common Model Names (Variants)

  • S 450 / S 500 / S 580 → petrol + mild hybrid
  • S 350d / S 400d → diesel
  • S 580e → plug-in hybrid
  • Maybach S 680 → ultra-luxury V12
  • AMG S 63 E PERFORMANCE → top performance

Key Features

  • MBUX Hyperscreen (giant curved glass display, optional)
  • Rear seat entertainment system with touchscreens
  • Massaging, heated, ventilated seats (front and rear)
  • 4D Burmester® sound system
  • AIRMATIC or E-ACTIVE BODY CONTROL suspension
  • Level 3 semi-autonomous driving (in some markets)
  • Soft-close doors, ambient lighting, fragrance system

Suitable for: ultimate comfort, luxury, status, and technology.
